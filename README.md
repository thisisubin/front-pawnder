# 🐾 Pawnder - Frontend

> 유기동물을 위한 입양·제보·커뮤니티 플랫폼의 **프론트엔드** 프로젝트입니다.

---

## 🎯 소개

**Pawnder**는 유기동물과 사람을 연결해주는 웹 서비스입니다.  
사용자는 유기견/유기묘를 제보하거나 입양 신청할 수 있으며,  
입양 후기와 반려동물 이야기를 나눌 수 있는 커뮤니티 공간도 함께 제공합니다.

이 저장소는 **React 기반의 프론트엔드 코드**로, 사용자 인터페이스(UI)와 클라이언트 로직을 담당합니다.

---

## 🖥 기술 스택

- **React**
- **JavaScript (ES6+)**
- **Vite**
- **React Router DOM**
- **Axios**
- **Styled Components**
- **ESLint / Prettier**

---

## 🚀 실행 방법

1. 의존성 설치
    ```bash
    npm install
    ```

2. 개발 서버 실행
    ```bash
    npm run dev
    ```

3. 브라우저에서 확인
    ```
    http://localhost:3000
    ```
